# Job Listings App

## Tools Used

1. create-react-app
2. material-ui
3. fetch
4. redux and redux saga

## Backend

- [json-server package](https://www.npmjs.com/package/json-server)
- Data item shape:

```JSON
{
    "id": 9,
    "title": "HR Specialist",
    "description": "Experienced with HR policies and systems.",
    "employment_type": "Full time",
    "location": "Alexandria"
}
```
